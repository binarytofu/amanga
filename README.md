# amanga

> 下载漫画咯

## Install

```bash
$ npm install -g @shayin/amanga
```

Or Yarn:

```bash
$ yarn global add @shayin/amanga
```

## Usage

```
$ amanga --help

    下载漫画咯

    Usage
        $ amanga <...input>

    Options
        --type       source site
        --info       print title and images list
        --output-dir the ouput directory  [default: amanga/<type>/<id>]

    Examples
        $ amanga 114883 --type nhentai
        $ amanga 114883 --type nhentai --info
```
